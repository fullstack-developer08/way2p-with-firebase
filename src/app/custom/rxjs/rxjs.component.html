of is used to convert object in the form of Observable.

  • Emit variable amount of values in a sequence and then emits a complete notification.

    const person = {{'{'}}

    const person$: Observable<Object> = of(person);

from is used to convert promise and observable into the observable.

  • from operator can be used to convert a promise to an observable!
  • For arrays and iterables, all contained values will be emitted as a sequence!
  • This operator can also be used to emit a string as a sequence of characters!
  • of operator can take multiple arguments, but from operator take one argument promise, array or string.
Example - 1 
    const employees = [
        {{'{'}}name: 'xyz1'},
        {{'{'}}name: 'xyz2'}
    ];


    const promiseOfEmployees: Promise<Object> = Promise.resolve(employees);
    const promiseOfEmployees$: Observable<Object> = from(promiseOfEmployees);

    promiseOfEmployees$.subscribe(e => console.log(e));

    // convertering oberservable into observertion(it has no uses in real time)

    const person2$: Observable<Object> = from(person$);

    person2$.subscribe(e => console.log(e));
Example - 2 (Converting string into sequence of char) 
from('string').subscribe(s=>console.log(s))
Example - 3 (Converting array into sequence of values) 
from([1,2,3,4,5,6,7,8]).subscribe(s=>console.log(s))

interval operator

Creates an Observable that emits sequential numbers every specified interval of time.

interval operator needs only one argument, that is interval of time.

Example 
import {{'{'}} interval } from 'rxjs';
import {{'{'}} take } from 'rxjs/operators';

const numbers = interval(1000);

const takeFourNumbers = numbers.pipe(take(4));

takeFourNumbers.subscribe(x => console.log('Next: ', x));

// Logs:
// Next: 0
// Next: 1
// Next: 2
// Next: 3

timer operator

Creates an Observable that starts emitting after an dueTime and emits ever increasing numbers after each period of time thereafter.

It is same like interval but it takes two arguments, first is dueTime and second is interval of time.

If we will provide only one argument that is dueTime then it will take only 1 element from the observable.

Example - 1

Emits ascending numbers, one every second (1000ms), starting after 3 seconds 

import {{'{'}} timer } from 'rxjs';

const numbers = timer(3000, 1000);
numbers.subscribe(x => console.log(x));
Example - 2

Emits one number after five seconds 

import {{'{'}} timer } from 'rxjs';

const numbers = timer(5000);
numbers.subscribe(x => console.log(x));

concat operator

Creates an output Observable which sequentially emits all values from given Observable and then moves on to the next.

Example 
import {{'{'}} concat, interval } from 'rxjs';
import {{'{'}} take } from 'rxjs/operators';

const timer1 = interval(1000).pipe(take(10));
const timer2 = interval(2000).pipe(take(6));
const timer3 = interval(500).pipe(take(10));

const result = concat(timer1, timer2, timer3);
result.subscribe(x => console.log(x));

// results in the following:
// (Prints to console sequentially)
// -1000ms-> 0 -1000ms-> 1 -1000ms-> ... 9
// -2000ms-> 0 -2000ms-> 1 -2000ms-> ... 5
// -500ms-> 0 -500ms-> 1 -500ms-> ... 9

{{tabs[5]}}

merge operator

Creates an output Observable which concurrently emits all values from every given input Observable.

Example 
import {{'{'}} merge, interval } from 'rxjs';
import {{'{'}} take } from 'rxjs/operators';

const timer1 = interval(1000).pipe(take(10));
const timer2 = interval(2000).pipe(take(6));
const timer3 = interval(500).pipe(take(10));

const result = merge(timer1, timer2, timer3);
result.subscribe(x => console.log(x));

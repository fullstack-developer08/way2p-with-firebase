Lombok Project

Introduction
Lombok is one of the most useful development tool/dependencies. Using lombok we can avoid boiler plate code for POJO class like setter, getter and no/all args constructor code.
Download Lombok and Add it to your Eclipse/STS/Any Editor

We can download the lombok jar from the lombok website.

After download we can run the jar using below command

java -jar lombok.jar

It will open below window and we need to provide our editor location. Once we done this we need to restart our editor.

Add lombok dependency to your project lombok pom.xml file 
<dependency>
    <groupId>org.projectlombok</groupId>
    <artifactId>lombok</artifactId>
    <optional>true</optional>
</dependency>
Now we can use below annotation in our pojo class.
@Data - This will add all the things like Setter, Getter, No Args Constructor, All Args Constructor etc..
@Getter - It will add only getter
@Setter - It will add only setter
@NoArgsConstructor - It will no args constructor, which is mandatory while working with JPA
@AllArgsConstructor - It will add all args constructor.

Example
package com.way2p.model;

import org.springframework.data.annotation.Id;

import lombok.AllArgsConstructor;
import lombok.Data;
import lombok.NoArgsConstructor;

@Data
@NoArgsConstructor
@AllArgsConstructor()
public class Customer {{'{'}}
    @Id
    private Long id;
    private String firstName;
    private String lastName;
    public Customer(String firstName, String lastName) {{'{'}}
        this.firstName = firstName;
        this.lastName = lastName;
    }

}
